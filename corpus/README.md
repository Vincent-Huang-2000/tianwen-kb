# 天文学实体标注体系 v1.0

基于 shiji-kb 的 〖TYPE 实体〗 标注格式，针对天文志内容设计。

## 八类天文实体

| 标记 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `〖星 X〗` | 恒星名 | 有专名的单颗恒星 | 〖星 北辰〗、〖星 织女〗、〖星 天枢〗、〖星 北落〗 |
| `〖官 X〗` | 星官/星座 | 恒星星组、二十八宿 | 〖官 北斗七星〗、〖官 紫微垣〗、〖官 角〗、〖官 房〗 |
| `〖曜 X〗` | 行星 | 五大行星 + 日、月 | 〖曜 岁星〗、〖曜 荧惑〗、〖曜 太白〗、〖曜 日〗 |
| `〖象 X〗` | 天象 | 异常天象、妖星 | 〖象 日食〗、〖象 彗星〗、〖象 流星〗、〖象 天棓〗 |
| `〖神 X〗` | 神祇/帝 | 天神、上帝 | 〖神 太一〗、〖神 五帝〗、〖神 天一〗 |
| `〖邦 X〗` | 邦国/地域 | 分野对应的政权 | 〖邦 秦〗、〖邦 楚〗、〖邦 齐〗 |
| `〖州 X〗` | 九州/分野 | 分野地域 | 〖州 雍州〗、〖州 冀州〗、〖州 扬州〗 |
| `〖度 X〗` | 度量 | 度数、距离、尺寸 | 〖度 五度〗、〖度 三十度〗、〖度 四丈〗 |

## 与 shiji-kb 18类的对应关系

| shiji-kb | 天文 | 说明 |
|----------|------|------|
| `〖@X〗` 人名 | `〖神 X〗` | 天文志中的人名多为神祇 |
| `〖=X〗` 地名 | `〖邦 X〗` `〖州 X〗` | 细分为政权和分野区域 |
| `〖;X〗` 官职 | — | 天文志极少出现 |
| `〖%X〗` 时间 | — | 保留用于事件阶段 |
| `〖!X〗` 天文 | `〖星 X〗` `〖官 X〗` `〖象 X〗` | 拆分为三类 |
| `〖$X〗` 数量 | `〖度 X〗` | 转化为天文度量 |

## 标注原则

1. **星官优先于恒星**：如果一组星有集体名称，整体标为星官；其中被单独命名的星标为恒星名
   - 例：〖官 北斗七星〗中〖星 天枢〗
2. **二十八宿标为星官**：角、亢、氐、房、心、尾、箕等均为星官
3. **行星与恒星区分**：五行星（岁星/荧惑/填星/太白/辰星）标为曜；其他标为星
4. **重复出现均标注**：同一实体在文中每次出现都标
5. **不标注标题和结构标记**
